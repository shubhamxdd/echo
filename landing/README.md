# Echo Landing Page

This folder contains the source code for the Echo API Client landing page.

## Structure
- `index.html`: The main landing page, built with Tailwind CSS v4 (via CDN) and Lucide Icons.

## How to View
Simply open `index.html` in any modern web browser.

## Design
- **Theme**: Dark Slate / Zinc-950
- **Accent**: Orange (Primary: `#f97316`)
- **Typography**: Geist (Sans), JetBrains Mono (Mono)
